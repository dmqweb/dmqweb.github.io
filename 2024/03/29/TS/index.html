# interface 和 type 的区别

1、实现对象和类一般使用 interface，

而实现联合类型、元组类型、交叉类型一般使用 type

2 、 interface 可以多次定义 并被视为合并所有声明成员 type 不支持

3 、 type 能使用 in 关键字生成映射类型，但 interface 不行。

4 、 默认导出方式不同
